# Clean Cep Validator

## Installation
---
```npm i clean-cep-validator```

<br/>

## Usage
---
#### JavaScript

```javascript
const { CepValidator } = require('clean-cep-validator')

const isValid = CepValidator.validate('12345-678') // true
```

#### TypeScript
```typescript
import { CepValidator } from 'clean-cep-validator'

const isValid = CepValidator.validate('12345-678') // true
```
